# Customer Churn Prediction

## Dataset Description
This project uses a **Customer Churn dataset** obtained from **Kaggle**. The dataset contains customer demographic information, usage details, and billing features. It is used to analyze and predict customer churn.

## Machine Learning Problem
The goal is to predict whether a customer will churn (stop subscribing) based on their information. This is a **classification problem** with the target variable **Churn**.

## Contents
- **Jupyter Notebook**: `customer_churn.ipynb`
- **Methodology Diagram**: `methodology_diagram.png`
- **Dataset**: `customer_churn.csv`

## How to Run the Code
1. Download the dataset and place it in the same directory as the Jupyter Notebook.
2. Open the `customer_churn.ipynb` in **Jupyter Notebook** or **Google Colab**.
3. Run the notebook.
